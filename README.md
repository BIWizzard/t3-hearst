# Hearst Data & Analytics SOW2

**Client:** Hurst  
**Project Type:** Trace Three Data & Analytics  
**Timeline:** April 2026 - December 2026  
**Status:** SOW Kickoff (scheduled: April 13, 2026)

## Overview

This project represents the second phase of engagement, building on a successful MVP round. We're modernizing Hurst's data architecture and migrating from their current Decentrix vendor solution to a contemporary, cost-effective platform.

## Project Scope

### Current State
- **Data Sources:**
  - AdSense
  - Wide Open (to verify naming)
- **Current Data Warehouse:** Decentrix

### Target Architecture
- Modern cloud-based data architecture
- Microsoft Fabric integration
- Accessible via Power BI analytics layer
- Data agent layer for advanced querying
- **Critical Constraint:** Decentrix must be sunset by January 1, 2027

### Key Deliverables
This is a build, not a migration:
- New data pipelines
- Medallion architecture implementation
- Data landing zone in Microsoft Fabric
- Power BI integration
- Data agent infrastructure
