## 2. Annex A — Project Specifications

### 2.1 Background

**Client Background.** Hearst Television is a national broadcast media company that previously engaged Trace3 for an initial data modernization MVP (referenced here as "Phase 0" — see `sow-v7-20251118.md`), which delivered a production-grade Microsoft Fabric environment and an AI-powered Pacing dashboard for sales forecasting. Building on that success, Client is extending the engagement to complete the full data platform modernization and expand AI capabilities across the organization.

**Business Objectives:**
- Retire the legacy **Decentrix** platform
- Expand conversational **AI agent** capabilities across Sales, Finance, and Ad Ops domains
- Deliver self-service analytics and unified **Power BI** dashboards to business stakeholders

**Technical Objectives:**
- Complete migration of **Decentrix** and **WideOrbit** ETL pipelines into **Microsoft Fabric Lakehouse**
- Establish **DEV / TEST / PROD** environments with **CI/CD**
- Expand the existing **Fabric AI Agent framework** into additional business domains

### 2.2 In-Scope

**In-Scope Software:**
- Microsoft Fabric (Lakehouse, Pipelines, OneLake)
- Power BI
- WideOrbit
- AdBook
- Decentrix (migration source)
- Azure OpenAI / Fabric AI Agent framework
- Purview (or Client-preferred governance tooling)

**In-Scope Environments:**
- DEV, TEST, and PROD Microsoft Fabric environments
- CI/CD pipeline configuration
- Power BI workspace governance

**Work Locations:**
- Onsite at 3540 Toringdon Way, Charlotte, NC 28277 (for key milestones and stakeholder sessions)
- Remote for work not requiring onsite presence

### 2.3 Phases

#### Plan Phase (1 week)
- Kickoff prerequisites, attendees, environment documentation, prep checklists
- Kickoff meeting: validate scope, governance, schedule, change request process, contacts
- Create Project Plan + Schedule
- Align on goals, success criteria, team structure, and sprint cadence under the **DTaaS (Data Transformation as a Service) delivery model**

#### Discovery Phase (1 week)
- Assess current state of:
  - Decentrix architecture
  - WideOrbit ETL pipelines
  - Existing Power BI assets
  - MVP Fabric environment
- Identify gaps, data quality issues, and AI use case opportunities

#### Design Phase (1 week)
- Design future-state Microsoft Fabric Lakehouse architecture including:
  - DEV / TEST / PROD environment topology
  - CI/CD pipeline framework
  - Power BI Row-Level Security (RLS) and AD Group structure
  - Expanded AI Agent domain model with prompt guardrails and intent mapping

#### Execute Phase (30 weeks) — "TWINS" DTaaS
Ongoing, client-directed engagement with a monthly budget envelope:

- **160 hours/month** of data engineering / data analyst services (daily and weekly tasks as needed)
- **20 hours/month** of data architect services
  - Pipeline architecture design across multiple data sources
  - ETL process development (APIs, databases, external providers)
  - Pipeline performance monitoring + troubleshooting
  - Implementation of data architecture best practices
  - Collaboration with Client data engineering / BI teams to translate business requirements into technical solutions
- **General project management** as needed

**Initiatives directable during Execute** (as prioritized by Client):
- WideOrbit ETL migration to Fabric
- Historical data migration and validation
- Decentrix decommission
- Fabric Lakehouse optimization
- DEV/TEST/PROD environment setup
- Fabric pipeline productionization
- MVP Power BI dashboards moved to PROD
- Unified Power BI views
- Sales AI Agent MVP
- Conversational Q&A Agent build

#### Closure Phase (1 week)
- User Acceptance Testing (UAT) with business + technical stakeholders
- Finalize as-built documentation
- Knowledge transfer sessions covering architecture, agent extension patterns, operational runbooks
- Hypercare support period post-deployment
- Completion Certificate + Client sign-off

> **Note on Knowledge Transfer:** Does not replace formal system training. Delivered as **one (1) session lasting up to two (2) hours**. Informal, not prepared in advance, no knowledge artifacts (docs/diagrams/videos) produced. Session may be recorded at Client's request and distributed within 3 business days.
